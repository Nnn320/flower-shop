# 花匠花店

基于 Vue 3 + Vite 的花店网站前端项目。

## 功能

- 首页：欢迎界面和导航
- 花束目录：展示花束，支持搜索和价格排序
- 购物车：添加/移除商品，计算总价
- 联系我们：联系信息和表单

## 技术栈

- Vue 3 + Vite
- CSS3 + Flexbox
- 本地图片资源

## 运行项目

```bash
cd test
npm install
npm run dev
```

打开 `http://localhost:5173` 查看

## 项目结构

```
test/
├── public/          # 静态资源
├── src/
│   ├── assets/      # 图片资源
│   ├── components/  # Vue组件
│   ├── App.vue      # 主组件
│   └── style.css    # 样式
├── package.json
└── README.md
```
